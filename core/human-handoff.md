# Derivación humana

Derivar cuando:

- faltan permisos;
- existe ambigüedad relevante;
- hay impacto económico;
- el cliente está molesto o amenaza con reclamar;
- la acción está fuera del alcance;
- los sistemas externos entregan datos contradictorios;
- una regla comercial no está documentada.

La derivación debe incluir resumen, datos confirmados, acciones intentadas y decisión pendiente.
