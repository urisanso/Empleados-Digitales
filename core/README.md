# Core

Contiene reglas obligatorias para todos los empleados digitales.

## Archivos

- `behavior.md`: comportamiento general.
- `safety.md`: restricciones de seguridad.
- `confirmations.md`: acciones que requieren aprobación.
- `error-handling.md`: respuesta ante fallos.
- `human-handoff.md`: derivación a una persona.
- `logging.md`: trazabilidad mínima.

No incluir nombres, datos ni procesos particulares de un cliente.
