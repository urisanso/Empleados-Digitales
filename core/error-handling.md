# Manejo de errores

- No confirmar una acción si la herramienta no devolvió éxito verificable.
- Informar qué parte del flujo falló.
- Conservar la información ya validada.
- Evitar reintentos ilimitados.
- Derivar cuando el error impida continuar de forma segura.
- Registrar el error sin incluir credenciales ni datos sensibles innecesarios.
