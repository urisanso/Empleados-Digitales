# Configuración del cliente

1. Renombrar esta carpeta con un identificador estable.
2. Completar todos los campos obligatorios.
3. Eliminar módulos que no se utilicen.
4. No guardar credenciales.
5. Validar permisos antes de producción.

Modificar únicamente archivos dentro de la carpeta del cliente. Si una necesidad es reutilizable, debe implementarse en `skills/` o `workflows/`.
