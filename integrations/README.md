# Integraciones

Esta carpeta documenta contratos y adaptadores para aplicaciones externas.

Cada integración debe especificar:

- proveedor;
- autenticación;
- permisos mínimos;
- operaciones permitidas;
- límites;
- errores esperables;
- estrategia de reintento;
- datos sensibles;
- método de prueba.

Las credenciales deben estar fuera del repositorio.
