# Convenciones

Usar minúsculas y guiones.

Las skills se nombran por capacidad:

```text
crm-read
sales-follow-up
payment-status
response-drafting
```

Los workflows se nombran por proceso:

```text
review-open-opportunities
prepare-sales-reply
detect-overdue-follow-ups
verify-pending-payment
```

No nombrar módulos por cliente cuando la función puede reutilizarse.
