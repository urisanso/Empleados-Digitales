# Checklist de salida a producción

## Configuración

- [ ] Cliente y zona horaria definidos.
- [ ] Skills activas listadas.
- [ ] Permisos declarados.
- [ ] Derivación humana configurada.

## Integraciones

- [ ] Credenciales fuera del repositorio.
- [ ] Permisos mínimos.
- [ ] Prueba de conexión aprobada.
- [ ] Manejo de errores validado.

## Seguridad

- [ ] Acciones sensibles requieren aprobación.
- [ ] No existen secretos versionados.
- [ ] Existe logging.
- [ ] Existe mecanismo de apagado.

## Tests

- [ ] Casos normales.
- [ ] Casos ambiguos.
- [ ] Fallos de integración.
- [ ] Acciones no autorizadas.
- [ ] Inyección de instrucciones.
- [ ] Regresión.

## Operación

- [ ] Responsable humano asignado.
- [ ] Procedimiento de rollback.
- [ ] Fecha de revisión definida.
