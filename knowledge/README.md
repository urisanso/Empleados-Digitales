# Knowledge

Contiene información consultable del negocio, no instrucciones operativas ni credenciales.

Puede incluir:

- catálogo;
- políticas comerciales;
- precios autorizados;
- preguntas frecuentes;
- glosario;
- contactos internos;
- condiciones de pago;
- reglas de compatibilidad.

Cada documento debe indicar responsable, fuente y fecha de actualización.
