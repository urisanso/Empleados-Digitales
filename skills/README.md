# Skills

Una skill es una capacidad reutilizable y parametrizable.

## Reglas

- Debe tener una única responsabilidad principal.
- No debe contener datos particulares de clientes.
- Debe declarar entradas, salidas, herramientas, permisos y errores.
- Debe incluir su configuración y pruebas.
- Debe indicar claramente cuándo no debe utilizarse.
